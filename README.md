# Valpiccola Frontend

Basic README how to run the frontend of Valpiccola

## Initial installation
```bash
yarn install
```

## Environmental variables
```
PUBLIC_SUPABASE_ANON_KEY=
PUBLIC_SUPABASE_URL=
PUBLIC_APP_HTTP_URL=
PUBLIC_API_HTTP_URL=
PUBLIC_SANITY_CDN=
PUBLIC_SANITY_PROJECT_ID=
PUBLIC_SANITY_TOKEN=
PUBLIC_STADIA_API_KEY=
PUBLIC_STADIA_URL=
```

## Update packages
```bash
yarn yarn-upgrade-all 
```

## Run staging
```bash
yarn run staging
```
